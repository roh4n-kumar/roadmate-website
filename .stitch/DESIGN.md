# RoadMate Premium Design System

## 1. Visual Philosophy
- **Aesthetic**: Modern Minimalism meets Luxury Performance.
- **Key Characteristics**: Glassmorphism, generous whitespace, high-fidelity typography, and smooth micro-animations.
- **Atmosphere**: "Clean, minimal, with generous whitespace and high-contrast typography." (Modern/Luxury)

## 2. Color Palette
- **Primary**: Deep Obsidian (#0F172A) - Backgrounds, primary text.
- **Accent**: Crimson Red (#E11D48) - Brand logo, CTA buttons, active states.
- **Surface**: Pure White (#FFFFFF) - Cards, Navbar (Glass effect).
- **Secondary**: Slate Gray (#64748B) - Secondary text, borders.
- **Background**: Soft Gray (#F8FAFC) - Page body.

## 3. Typography
- **Headings**: 'Inter', sans-serif (Font-weight: 900) - Tight letter-spacing (-2px).
- **Body**: 'Inter', sans-serif (Font-weight: 400, 500).
- **Buttons**: 'Inter', sans-serif (Font-weight: 800) - Uppercase/Bold.

## 4. Components
- **Buttons**: Pill-shaped, Crimson Red background, white text, subtle hover lift.
- **Cards**: Softly rounded (16px), whisper-soft shadows, thin 1px borders (#E2E8F0).
- **Navbar**: Constant 70px height, White Glassmorphism (blur 15px), Slate borders.

## 5. Design System Notes for Stitch Generation
- **Radius**: `rounded-2xl` (16px)
- **Shadows**: `shadow-subtle` (low opacity, wide spread)
- **Vibe**: High-end luxury automotive feel.
