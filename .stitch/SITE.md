# RoadMate - Premium Rental Platform context

## Vision
An "Award Winning" premium online bike and car rental website. The UI should evoke a sense of luxury, trust, and speed.

## Stacks
- React + Vite
- Firebase Authentication
- Framer Motion (Animations)
- FontAwesome / React Icons
- Custom CSS (Premium Glassmorphism & Micro-interactions)

## Sitemap
1. [ ] **Homepage** (`Home.jsx` / `Hero.jsx`) - High-impact luxury hero, vehicle categories.
2. [ ] **Search Results** (`VehicleResults.jsx`) - Premium card grid with filter sidebar.
3. [ ] **Vehicle Details** (`Booking.jsx` / `BookingForm.jsx`) - Elegant details page with trust markers.
4. [ ] **User Auth** (`Login.jsx` / `Signup.jsx`) - Clean, minimalist premium authentication.
5. [ ] **My Bookings** (`MyBookings.jsx`) - Sophisticated dashboard for managing rentals.
6. [ ] **Profile/Settings** (`PersonalInfo.jsx`) - Secure and sleek account management.

## Roadmap
- **Iteration 1**: Global Design System & Homepage Hero.
- **Iteration 2**: Vehicle Card Grid & Search UI.
- **Iteration 3**: Forms & Authentication UI.
- **Iteration 4**: Dashboard & Verification Flow.
